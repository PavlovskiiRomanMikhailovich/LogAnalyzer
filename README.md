# LogAnalyzer

Программа для анализа лог-файлов.  
Программа считает:

- общее количество строк в файле  
- количество ошибок (`ERROR`)  
- топ-3 пользователей по количеству ошибок  

## Запуск

### Запуск программы (c указанием пути к файлу "sample.log")
```bash
npm start ./sample.log
```

### Запуск тестов

```bash
npm test
```

## Крайние случаи и ограничения

Программа учитывает и корректно обрабатывает следующие ситуации:

- Не указан путь к файлу или файл не найден  
- Нет доступа к файлу или путь ведёт на папку  
- Файл пустой или слишком большой (>50 MB)  
- Строки с неверным форматом считаются битым (`invalidCount`)  
- Пустые строки игнорируются 

## Пример формата логов

Логи должны иметь следующий формат:

`2024-12-01T12:00:49Z INFO user=48 action=checkout`

`2024-12-01T12:00:50Z ERROR user=43 action=payment`
